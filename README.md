This project is an interactive Microsoft Power BI dashboard built to analyze Airbnb listing data and generate insights about pricing, room types, neighbourhood trends, and customer reviews. The dashboard helps understand how different factors impact Airbnb listings and customer preferences across various locations.

The analysis includes key business insights such as average price by neighbourhood, review distribution by room type, percentage share of different room categories, and geographical visualization of listings using maps. Interactive filters and visualizations allow users to explore trends and compare listing performance across different areas.

Price analysis by neighbourhood
Reviews analysis by room type
Distribution of different room types
Neighbourhood-wise listing map
