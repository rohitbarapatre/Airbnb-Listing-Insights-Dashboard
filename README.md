# Airbnb Listings Analysis Dashboard – Power BI Project

## Overview
This project is an interactive Microsoft Power BI dashboard developed to analyze Airbnb listing data and generate valuable business insights related to pricing trends, room types, neighbourhood performance, and customer reviews. The dashboard provides a comprehensive visualization of Airbnb operations and helps understand how different factors influence customer preferences and listing performance across various locations.

The project focuses on transforming raw Airbnb dataset information into meaningful visual insights using data cleaning, business analytics, and dashboard storytelling techniques. Interactive filters, maps, charts, and KPI visuals allow users to explore listing trends dynamically and compare neighbourhood performance effectively.

---

# Problem Statement
Airbnb platforms contain thousands of listings spread across different locations, making it difficult for businesses and hosts to analyze customer preferences, pricing trends, and room availability manually. Without proper visualization and analysis, identifying profitable areas and customer behavior patterns becomes challenging.

This project aims to solve these challenges by:
- Analyzing Airbnb listing trends
- Identifying high-performing neighbourhoods
- Understanding pricing variations across locations
- Comparing room type popularity
- Visualizing customer review distribution
- Supporting data-driven business decisions using interactive dashboards

---

# Project Objectives
- Build an interactive Power BI dashboard for Airbnb data analysis
- Perform neighbourhood-wise pricing analysis
- Analyze customer reviews by room type
- Visualize listing distribution using maps
- Compare percentage share of room categories
- Create visually appealing and user-friendly reports
- Improve business understanding through analytical storytelling

---

# Dashboard Features
The dashboard contains multiple visualizations and interactive components designed for detailed exploration of Airbnb data.

## Key Dashboard Components
### 1. Price Analysis by Neighbourhood
- Displays average Airbnb prices across different neighbourhoods
- Helps identify premium and budget-friendly locations
- Enables comparison of pricing trends between areas

### 2. Reviews Analysis by Room Type
- Shows customer review distribution for different room categories
- Helps understand customer engagement and satisfaction trends

### 3. Room Type Distribution
- Displays percentage share of:
  - Entire Home/Apartment
  - Private Room
  - Shared Room
- Helps identify the most commonly available accommodation types

### 4. Geographical Listing Visualization
- Interactive map visualization of Airbnb listings
- Displays neighbourhood-wise property concentration
- Helps identify high-demand locations visually

### 5. Interactive Filters & Slicers
Users can dynamically filter data based on:
- Neighbourhood
- Room Type
- Price Range
- Review Scores
- Availability

# Data Analysis Process

## 1. Data Collection
The Airbnb dataset was imported from Excel/CSV files containing listing details such as:
- Listing ID

- Room Type
- Price
- Reviews
- Availability
- Latitude & Longitude

## 2. Data Cleaning
The dataset was cleaned using:
- Missing value handling
- Duplicate removal
- Data formatting
- Null value treatment

## 3. Data Transformation
Data transformation steps included:
- Creating calculated measures
- Aggregating neighbourhood pricing
- Categorizing room types
- Generating KPIs for dashboard analysis

## 4. Data Visualization
Different Power BI visualizations were used:
- Bar Charts
- Pie Charts
- KPI Cards
- Maps
- Line Charts
- Slicers & Filters
